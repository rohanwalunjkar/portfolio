## How to Host on GitHub Pages

### 1. Create a public repo
On GitHub, create a new **public** repository named `portfolio` (do NOT add a README when creating it, you already have one).

### 2. Push your files
Open a terminal inside this `portfolio` folder and run:

```bash
git init
git add .
git commit -m "Add portfolio website"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/portfolio.git
git push -u origin main
```

### 3. Turn on GitHub Pages
1. Go to your repo on GitHub.
2. Click **Settings**.
3. Click **Pages** (left sidebar).
4. Under **Source**, choose **Deploy from a branch**.
5. Branch: **main**, Folder: **/ (root)**.
6. Click **Save**.
7. Wait 1-2 minutes. Site goes live at `https://YOUR-USERNAME.github.io/portfolio/`

### Optional: make it your main site
Name the repo exactly `YOUR-USERNAME.github.io` and your URL becomes `https://YOUR-USERNAME.github.io/`
